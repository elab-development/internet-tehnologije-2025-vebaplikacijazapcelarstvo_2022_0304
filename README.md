# Aplikacija za Pčelarstvo

Veb aplikacija za upravljanje pčelinjim društvima, košnicama i aktivnostima.

## O Projektu

Aplikacija omogućava pčelarima da:
- Evidenciju košnica sa detaljnim informacijama
- Planiranje i praćenje aktivnosti
- Dodavanje komentara na aktivnosti
- Upravljanje notifikacijama
- Razlikovanje korisnika po ulogama (ADMIN, KORISNIK, MENADŽER)

---

## Tehnologije

### **Frontend:**
- **Next.js 16.1.3** - React framework
- **React 19.2.3** - UI biblioteka
- **TypeScript 5** - Tipizacija
- **Tailwind CSS 4** - Stilizovanje
- **@remixicon/react** - Ikonice

### **Backend:**
- **Next.js API Routes** - RESTful API
- **Prisma ORM 5.22.0** - Database ORM
- **MySQL** - Relaciona baza podataka

### **Autentifikacija:**
- **JSON Web Tokens (jsonwebtoken)** - JWT autentifikacija
- **bcryptjs** - Heširanje lozinki
